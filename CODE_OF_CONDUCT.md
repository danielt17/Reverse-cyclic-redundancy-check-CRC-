# Code of Conduct

## Our standards

Participants are expected to:

- Be respectful and constructive.
- Focus on technical discussion and collaboration.
- Accept feedback in good faith.
- Avoid harassment, discrimination, and personal attacks.

Unacceptable behavior includes:

- Insults, threats, or intimidation.
- Trolling or deliberately disruptive behavior.
- Publishing private information without consent.

## Scope

This Code of Conduct applies to:

- GitHub issues and pull requests
- Discussions and review comments
- Any project-maintained communication channels

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and contributions that violate this Code of Conduct.

Repeated or severe violations may result in temporary or permanent bans from project participation.

## Reporting

For conduct concerns, open a private security-style report as described in `SECURITY.md` and label it as conduct-related.
